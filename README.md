# shotgame

this project is shotgame.

# using
1. install HSP
	goto http://hsp.tv/
	click & download & install from [Download now]
2. clone this repository
3. boot hsp editor & open shotgame.hsp
4. [Menu] -> [HSP] -> [実行ファイル自動作成]. or [ctrl] + [F9]
5. run shotgame.exe
